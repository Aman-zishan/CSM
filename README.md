### Temp setup

This CLI tool is under development and not yet published to npm 

```
npm run build
```

### help command

```
node dist/index.js -h
```

### save a code snippet

```
node dist/index.js -s Desktop/project/hello_world.py
```

### Retrieve snippet

```
node dist/index.js -l
```
